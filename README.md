# OLD POP 추천 서비스
## 1. 목표와 기능
### 1.1 목표
- 선택한 조건에 맞는 올드팝을 추천해주는 사이트 만들기

### 1.2기능
- 첫 번째 홈페이지에 나타나는 선택지 중 원하는 조건을 선택
- 선택한 조건에 맞는 올드팝의 제목을 가수, 발매 년도와 함께 출력
